CURP：多样化虚拟用户画像与回答生成平台

CURP: A platform for diverse virtual user profiling and response generation
GitHub / 代码仓库 arXiv / 论文
💡 Key Idea / 核心理念
用户由不同的 prototype 组合而成，用户的每个行为本质上折射出若干 prototype。
我们采用两阶段训练：第一阶段构造对应的 prototype codebook；第二阶段训练下游 LLM 理解与运用这些 prototype。
在 CURP 中，这些 prototype 被组织在 codebook 中，从而实现大规模、可控的用户模拟。

Users are composed of different prototypes, and each behavior reflects a mixture of prototypes.
We use a two-stage training pipeline: stage 1 builds the prototype codebook; stage 2 trains the downstream LLM to understand and use these prototypes.
CURP organizes prototypes in a codebook to enable scalable, controllable user simulation.
