✨ What can this demo do? / 这个 Demo 能做什么？
  • 基于 codebook 抽样生成多样化虚拟用户画像 / Sample diverse virtual users from the codebook.
  • 让同一问题得到多风格回答 / Generate varied answers to the same question.
  • 输出用户画像（Demographic）/ Generate demographic-style profiles.
  • 支持自定义单用户调控并对比多轮输出 / Customize a single user and compare rounds.
适用场景 / Use cases:
• 社会调查模拟 / Social survey simulation
• 用户研究预演 / User research rehearsal
• 内容生成多样性测试 / Content diversity testing
